抽屉 Drawer ​

触发命令后，从屏幕一侧滑出的抽屉式的面板。

基本用法 ​

点击触发按钮抽屉从右侧滑出，点击遮罩区关闭。

<template>
  <div class="box">
    <sb-drawer v-model:visible="visible" @ok="handleOk" @cancel="handleCancel" mask="false">
          <template #title>Title</template>
            <template #body>You can customize modal body text by the current situation. This modal will be closed
        immediately once you press the OK button.</template>
          </sb-drawer>
    <sb-button @click="handleClick">Open Drawer</sb-button>
  </div>
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const visible = ref(false)

const handleClick = () => {
  visible.value = true;
};
const handleOk = () => {
  visible.value = false;
};
const handleCancel = () => {
  visible.value = false;
}
</script>

抽屉位置 ​

自定义位置，设置 palcement 取值则抽屉从相应的位置滑出。

  
<template>
  <div class="box">
    <sb-drawer v-model:visible="visible" placement="bottom" @ok="handleOk" @cancel="handleCancel">
       <template #title>Title</template>
       <template #body>You can customize modal body text by the current situation. This modal will be closed immediately once you press the OK button.</template>
    </sb-drawer>
    <sb-button @click="handleClick ">Open Drawer</sb-button>
  </div>
</template>

<script lang="ts" setup>
import { ref } from 'vue';
  const visible = ref(false)

  const handleClick = () => {
    visible.value = true;
  };
  const handleOk = () => {
      visible.value = false;
  };
  const handleCancel = () => {
      visible.value = false;
  }
</script>

自定义节点 ​

通过设置 header、footer为false、hideCancel为true 来控制显示或隐藏。

<template>
  <div class="box">
      <sb-drawer :width="340" :header="false" :visible="visible" @ok="handleOk" @cancel="handleCancel">
          <template #title>
            Title
          </template>
          <template #body>You can customize modal body text by the current situation. This modal will be closed immediately once you press the OK button.</template>
      </sb-drawer>
      <sb-button @click="handleClick">Open Drawer</sb-button>
  </div>
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const visible = ref(false)
const handleClick = () => {
  visible.value = true;
};
const handleOk = () => {
  visible.value = false;
};
const handleCancel = () => {
  visible.value = false;
}
</script>

嵌套抽屉 ​

在抽屉内打开新的抽屉。

<template>
    <div class="box">
        <sb-button type="primary" @click="handleClick">Open Drawer</sb-button>
    <sb-drawer :visible="visible" :width="500" @ok="handleOk" @cancel="handleCancel" unmountOnClose>
        <template #title>
            Title
        </template>
        <template #body>
            <div :style="{ marginBottom: '20px' }">You can customize modal body text by the current situation. This modal
                will be closed immediately once you press the OK button.</div>
            <sb-button type="primary" @click="handleNestedClick">Open Nested Drawer</sb-button>
        </template>
    </sb-drawer>
    <sb-drawer :visible="nestedVisible" @ok="handleNestedOk" @cancel="handleNestedCancel" unmountOnClose>
        <template #title>
            Title
        </template>
        <template #body>
            <div>You can customize modal body text by the current situation. This modal will be closed immediately once
                you press the OK button.</div>
        </template>
    </sb-drawer>
    </div>
</template>
<script lang="ts" setup>
import { ref } from 'vue'

const visible = ref(false);
const nestedVisible = ref(false);
const handleClick = () => {
    visible.value = true;
};
const handleOk = () => {
    visible.value = false;
};
const handleCancel = () => {
    visible.value = false;
}
const handleNestedClick = () => {
    nestedVisible.value = true;
};
const handleNestedOk = () => {
    nestedVisible.value = false;
};
const handleNestedCancel = () => {
    nestedVisible.value = false;
}

</script>
