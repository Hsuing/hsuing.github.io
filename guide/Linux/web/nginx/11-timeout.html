й»ҳи®Өпјҡ60s

й…ҚеңЁпјҡhttpдёӯгҖҒserverдёӯгҖҒlocationдёӯ

  еңЁдёҺupstreamзҡ„serverе»әз«ӢиҝһжҺҘеҗҺпјҢе°ұдјҡжҠҠиҜ·жұӮеҫҖserverеҸ‘йҖҒпјҢиҝҷдёӘж—¶й—ҙжҳҜдёӨж¬Ўж•°жҚ®зҡ„еҸ‘йҖҒж—¶й—ҙе·®пјҢдёҚжҳҜж•ҙдёӘеҸ‘йҖҒиҝҮзЁӢзҡ„гҖӮжҜ”еҰӮиҜҙиҙҹиҪҪеӨ§гҖҒзҪ‘з»ңеҚЎпјҢеңЁtengineеҗ‘serverеҸ‘йҖҒиҜ·жұӮж—¶зӘҒз„¶еҚЎдәҶдёҖдёӢпјҢз„¶еҗҺз»§з»ӯеҸ‘йҖҒпјҢиҖҢиҝҷдёӨж¬Ўзҡ„ж—¶й—ҙе·®пјҲе…¶е®һе°ұжҳҜдёӨж¬Ўwriteзҡ„ж—¶й—ҙе·®пјүи¶…иҝҮдәҶжҲ‘и®ҫзҪ®зҡ„55sпјҢtengineе°ұдјҡи¶…ж—¶жҠҘй”ҷпјҢеҜ№дәҺиҝҷдёӘеҸӮж•°пјҢжҲ‘еҪ“еүҚдјҳеҢ–зҡ„жҳҜ55s
  • proxy_read_timeout 60s;

й»ҳи®Өпјҡ60s

й…ҚеңЁпјҡhttpдёӯгҖҒserverдёӯгҖҒlocationдёӯ

 еңЁе°ҶиҜ·жұӮеҸ‘йҖҒз»ҷupstreamзҡ„serverеҗҺпјҢеҗҺз«Ҝserverе°ұдјҡеӣһдј ж•°жҚ®пјҢиҝҷдёӘж—¶й—ҙжҳҜдёӨж¬Ўж”¶еҸ–ж•°жҚ®зҡ„ж—¶й—ҙе·®пјҢдёҚжҳҜж•ҙдёӘзҡ„жҺҘж”¶ж—¶й—ҙгҖӮжҜ”еҰӮиҜҙиҙҹиҪҪеӨ§гҖҒзҪ‘з»ңеҚЎпјҢеңЁз¬¬1ж¬Ўж”¶еҲ°иҜ·жұӮзҡ„ж•°жҚ®ж—¶ж–ӯдәҶпјҢз„¶еҗҺиҝҮдәҶ60sеҗҺжүҚж”¶еҲ°еҗҺйқўзҡ„ж•°жҚ®пјҢиҝҷдёӨдёӘж—¶й—ҙе·®(е…¶е®һе°ұжҳҜдёӨж¬Ўreadзҡ„ж—¶й—ҙе·®)и¶…иҝҮдәҶи®ҫзҪ®зҡ„60sпјҢtengineпјҲnginxпјүе°ұдјҡи¶…ж—¶жҠҘй”ҷпјҢжҲ‘еҪ“еүҚиө°зҡ„жҳҜй»ҳи®Өи®ҫзҪ®60s
  • keepalive_timeout 65s

й»ҳи®Өпјҡ75s

й…ҚеңЁпјҡhttpдёӯгҖҒserverдёӯгҖҒlocationдёӯ

  httpжҳҜж— зҠ¶жҖҒзҡ„еҚҸи®®пјҢеҪ“жңҚеҠЎз»“жқҹеҗҺпјҢе°ұйқўдёҙзқҖжҳҜеҗҰж–ӯејҖtcpиҝһжҺҘзҡ„й—®йўҳпјҢеҪ“е®ўжҲ·з«ҜжҲ–иҖ…жңҚеҠЎеҷЁз«ҜйңҖиҰҒж—¶пјҢеҸҜд»ҘеңЁе»әй“ҫзҡ„ж—¶еҖҷйҮҮз”Ёй•ҝиҝһжҺҘж–№ејҸпјҢеҚіжңҚеҠЎз»“жқҹеҗҺеңЁдёҖж®өж—¶й—ҙеҶ…дёҚж–ӯејҖиҝһжҺҘпјҢеҪ“еҶҚжңүиҜ·жұӮиҝҮжқҘж—¶зңҒжҺүдәҶе»әй“ҫзҡ„иө„жәҗж¶ҲиҖ—пјҢи¶…ж—¶еҗҺtengineпјҲnginxпјүдјҡдё»еҠЁж–ӯејҖиҝһжҺҘпјҢеҪ“з„¶й…ҚзҪ®йҮҢиҝҳжңүеҸҰеӨ–дёҖдёӘеҸӮж•° keepalive_requests 600;пјҢиҝҷдёӘеҸӮж•°жҳҜиҜҙеҚідҪҝй•ҝиҝһжҺҘжІЎеҲ°иҝҮжңҹж—¶й—ҙпјҢдҪҶжңҚеҠЎзҡ„httpжҖ»ж•°йҮҸи¶…иҝҮжҢҮе®ҡеҖјеҗҺд№ҹжҳҜиҰҒж–ӯејҖиҝһжҺҘпјҢжҲ‘зӣ®еүҚи®ҫзҪ®зҡ„жҳҜ600
  • resolver_timeout 10s

й»ҳи®Өпјҡ30s

й…ҚеңЁпјҡhttpдёӯгҖҒserverдёӯгҖҒlocationдёӯ

иҝҷдёӘжҳҜdnsи§Јжһҗи¶…ж—¶ж—¶й—ҙпјҢеҰӮжһңз”ЁдҪңжӯЈеҗ‘д»ЈзҗҶж—¶е°ұжңүз”ЁдәҶпјҢеҗҢж—¶еҸҜд»Ҙз”Ёresolver 127.0.0.1 valid=10m;жҢҮд»ӨжқҘжҢҮе®ҡdnsпјҢеҗҺйқўжҳҜи§ЈжһҗеҗҺзј“еӯҳзҡ„жңүж•Ҳж—¶й—ҙгҖӮ

server 127.0.0.1:9999 max_fails=20 fail_timeout=10s;

иҝҷдёӘжҳҜжҢҮжҹҗдёҖдёӘupstreamзҡ„serverеҰӮжһңеӨұиҙҘ20ж¬ЎеҗҺпјҢдёҚеҸҜд»Ҙж“ҚдҪңзҡ„ж—¶й—ҙпјҢй»ҳи®Өе°ұжҳҜ10sпјҢе…¶е®һеҸҜд»ҘеҸҰеӨ–зҡ„еҶҷжі•й…ҚеңЁhttpдёӯпјҢжҲ‘д№ жғҜзӣҙжҺҘй…ҚеңЁserverзҡ„еҗҺз«ҜгҖӮ

  • keepalive_timeout 65 70

иҝҷжҳҜеүҚз«Ҝkeepalive_timeoutзҡ„дёҖдёӘе»¶дјёй…ҚзҪ®пјҢеүҚйқў65жҳҜе‘ҠиҜүе®ўжҲ·з«ҜжҲ‘з»ҷдҪ дҝқжҢҒеӨҡд№…пјҢеҗҺйқўдёҖдёӘжҳҜеӨҡд№…жҲ‘е°ұз»ҷж–ӯејҖиҝһжҺҘдәҶ

и§ЈеҶіж–№ејҸ

1гҖҒеҰӮжһңзі»з»ҹе№¶еҸ‘йҮҸдёҚеӨ§пјҢжІЎжңүеҝ…иҰҒејҖеҗҜй•ҝиҝһжҺҘпјҢжңүдёӨз§Қж–№ејҸпјҢ

дәҢгҖҒз¬¬дәҢеҸ°nginxзҡ„keepalive_timeoutеҸҜд»Ҙй…ҚзҪ®дёә0пјҲй»ҳи®ӨжҳҜ75пјүгҖӮ

2гҖҒдёҠиҝ°й—®йўҳжҲ‘зҡ„и§ЈеҶіж–№жЎҲжҳҜпјҡжҡӮж—¶и°ғеӨ§keepalive_timeoutзҡ„еҖјпјҢе…Ҳи§ӮеҜҹпјҢдҪҶеҫҲжңүеҸҜиғҪиҝҳжҳҜдјҡжңүиҝҷдёӘй—®йўҳгҖӮ
